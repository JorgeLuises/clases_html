Encabezados (h1-h6)

Los encabezados en HTML van representados con la etiqueta < h1 >.
Define los elementos o textos más importantes (como titulos) de la pagína.

Ejemplo y sintaxis

< h1 > Encabezado < /h1 >

Area

La etiqueta < area >, define una sección para una imagen dentro de otra imagen principal a la que se le puede hacer clic e interactuar con ella.

Ejemplo y sintaxis

<area shape="" cords="" href="" alt="">

Iframe

La etiqueta < iframe >, se utiliza para poder incrustar algun documento u archivo dentro del actual codigo html utilizado.

Ejemplo y sintaxis

<iframe src=""></iframe>

HR

La etiqueta < hr >, se utiliza para realizar y definir cambios en el contenido html.
Es usualmente usado como regla para separar contendido del sitio web.

Ejemplo y sintaxis

<hr>
